Project Information

Objective, Technologies & Docker Strategy

This showcase highlights modern containerized deployment patterns for a Spring Boot student management platform.

Project Objective

Demonstrate how Docker containers simplify deployment, scalability, and environment consistency for a Java Spring Boot application connected to a MySQL database.

Technologies Used

Java 21 Spring Boot 3 REST API MySQL Docker Docker Compose Bootstrap 5

Docker Architecture Overview

Two lightweight containers operate together: the Spring Boot service exposes REST endpoints, while a MySQL container persists student data. Docker Compose orchestrates them.

Why Docker?

Docker ensures reproducible deployments, quick setup for new environments, seamless scaling, and reliable isolation of services during development and presentation.
